解答P9



  函式 find_max_position 用於找出單字出現次數最高的單字位置
  利用存放函式出現次數的 freq 陣列與存放總單字數量的整數 word_num，
  重複判斷目前最高單字出現次數
  迴圈結束後，印出出現次數最多的單字與其出現次數
  函式結束 

                        void find_max_position(int word_num, int freq[], int *max_p, char words[][15]) // 找出單字出現次數最高的單字位置
                        {
                            for (int i = 0; i < word_num; i++) // 當 i 小於單字總數迴圈持續執行，每執行一次 i 加 1
                            {
                                if(freq[i] > freq[*max_p]) // 判斷當前單字出現次數是否大於目前最高單字出現次數
                                {
                                    *max_p = i; // 將當前單字索引值存為目前出現次數最高的單字索引值
                                }
                            }
                            printf("%s:%d",words[*max_p] ,freq[*max_p]);
                        }

解答P10



  函式 count_freq 用於分別存放出現過的單字並計算單字出現頻率
  重複利用索引值 p 檢查文句 A 中出現的每個單字並判斷當下單字是否曾出現過
  若 p 目前位置不為空白則檢查當下單字是否等於先前任何單字
  若等於任一單字，則設定對應單字出現次數加 1
  若不等於任一單字，則將新單字加進存放出現過單字的 words 陣列
  函式結束，程式結束
