骰子遊戲 (進階題1)


A, B 兩人用二顆骰子計點比賽，一開始都是點數 1 朝上，點數 4 朝玩家，點數 2 朝右(骰子展開如下圖所示)。接下來 M 次修改操作，每次操作包含二個整數 x, y 代表兩個骰子操作。
x, y 意義為：1 表示向前滾一次。2 表示向右滾一次。操作修改完成，請輸出分數。

計分方式：
1. 若二顆點數均相同，計 0 分，例如 6, 6 或 3, 3。
2. 若二顆點數不同，加總二顆點數，例如 1, 2 (1+2=3 分)或 2, 6(2+6=8 分)。

傳值與傳址



  傳值：複製一份值傳入函式，不會更動到原本的變數
  傳址：複製位址傳入函式，這時函式將可以透過位址更改原本變數

                        #include <stdio.h>
                        void changenum_arr(int t[]);
                        void changenum_int(int t);

                        int main(){
                            int temp_arr[1] = {0};
                            int temp_int = 0;
                            
                            changenum_arr(temp_arr);
                            changenum_int(temp_int);

                            printf("%d\n", temp_arr[0]);
                            printf("%d", temp_int);
                        }

                        void changenum_arr(int t[]){ // 傳陣列傳址，會更動到原本的變數
                            t[0] = 10;
                        }

                        void changenum_int(int t){ // 傳變數傳值，不會更動到原本的變數
                            t = 10;
                        }
  output: //第一行印出 t[0]，第二行印出 t
  10
  0

                        void modify_dice(int dice_[][6], int modify_num, int modify_[][2]) //傳入玩家A或B的骰子，骰子修正次數，骰子修正方式
                        {
                            int dice_num;  // 0 表示第一顆骰子, 1 表示第二顆骰子
                            int mode; // 1 表示表示向前滾一次, 2 表示向右滾一次

                            for(int i = 0; i < modify_num; i++) // 修正幾次執行幾次迴圈
                            {   
                                for(int j = 0; j < 2; j++) //  2 顆骰子，所以執行 2 次迴圈
                                {
                                    dice_num = j; // 表示當前操作第一顆骰子或第二顆骰子
                                    mode = modify_mode(modify_, i, j); // 得到當前骰子要向前滾一次還是向右滾一次
                                
                                    if(mode == 1) // 向前滾一次
                                        roll_forward_1( dice_, dice_num);
                                    else if(mode == 2) // 向右滾一次
                                        roll_rigth_2( dice_, dice_num);
                                    else{}
                                }
                            }
                        }

解答P9



  使用函式 modify_dice 修正骰子點數
  宣告整數 dice_num, mode 表示當前骰子和當前操作方式
  根據修正次數有幾次執行幾次迴圈，每次都操作兩骰子的六面點數
  呼叫函式 modify_mode 得到當前操作方式是向前滾一次或向右滾一次
  根據當前操作方式操作骰子 

                        void modify_dice(int dice_[][6], int modify_num, int modify_[][2]) //傳入玩家A或B的骰子，骰子修正次數，骰子修正方式
                        {
                            int dice_num;  // 0 表示第一顆骰子, 1 表示第二顆骰子
                            int mode; // 1 表示表示向前滾一次, 2 表示向右滾一次

                            for(int i = 0; i < modify_num; i++) // 修正幾次執行幾次迴圈
                            {   
                                for(int j = 0; j < 2; j++) // 每次都需要修正 2 顆骰子，所以執行 2 次迴圈
                                {
                                    dice_num = j; // 表示當前操作第一顆骰子或第二顆骰子
                                    mode = modify_mode(modify_, i, j); // 得到當前骰子要向前滾一次還是向右滾一次
                                
                                    if(mode == 1) // 向前滾一次
                                        roll_forward_1( dice_, dice_num);
                                    else if(mode == 2) // 向右滾一次
                                        roll_rigth_2( dice_, dice_num);
                                    else{}
                                }
                            }
                        }

解答P10



  使用函式 modify_mode 回傳操作方式
  1 表示向前滾一次， 2 表示向右滾一次

                        int modify_mode(int modify_[][2], int count, int dice_num)
                        {
                            if(modify_[count][dice_num] == 1)
                                return 1; //向前滾一次
                            else if(modify_[count][dice_num] == 2)
                                return 2; //向右滾一次
                            else
                                return 0;
                        }
